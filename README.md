## 고령친화 도시 소개
###  노인이 건강하고 활력있는 사회생활을 영위할 수 있도록 정책 및 인프라, 서비스 등이 조성된 도시입니다. 궁극적으로 노인을 비롯한 전 세대가 다 함께 살기좋은 도시환경 조성을 지향합니다.
